### 追加機能
- 四則演算：＋以外の四則演算をする
- 上クリア：entry1に入力されている数字，数式をdeleteする
- 下クリア：entryに入力されている数字，数字をdeleteする
- オールクリア：entryとentry1に入力されている数字，数式の文字列全体をdeleteする
- 計算過程が見やすいようにもう一つのentry、entry1を作った。entry1には計算の様子が格納される
- 計算の演算子を連続で入れると書き換わるようにした
